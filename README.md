# Consultant Performance Dashboard

Interactive single-file HTML dashboard for the Intuit Money Payments team, built on data from the Sales Ops Insights Hub (Tableau).

## Features

- Global filters: Anaplan L3 business, Manager, Agent multi-select, and Week range
- KPI cards: self-serve toggle by category (Calls, Quality, Meetings, Payments, Chats)
- FTE / CW / Partner cohort cards: averages across active agents in current view
- Agent performance table: sortable, searchable, flat or grouped-by-manager view
- Auto-generated insights: productivity and outcome gaps called out automatically
- Week-over-Week tab: weekly trend charts, WoW delta KPI cards, and agent mover tables

## Usage

Open MoneyPayments_Dashboard.html directly in a browser. No server required, no dependencies to install. All chart rendering uses Chart.js loaded from CDN.

## Data

FY26 (Aug 1, 2025 - Jul 29, 2026). Data last captured from Tableau on May 13, 2026.
